----------------------INSTRUCTIONS-----------------------------------------
To run the programs:
Use Linux terminal to run the program using g++ compiler.
---------------------------------------------------------------------------
Run program2.cpp in one terminal first and then
program1.cpp in another terminal.

For compiling and running the program use:
g++ program1.cpp -o program1 and press enter.
and then type ./program1 and press enter.

Similarly for program2,replace program1 in the above commands.

After running the programs,program1 terminal will ask to enter the data
whereas program2 will simultaneously show the number of Integers
and Words in the given line.
---------------------------------------------------------------------------
To exit the program: Use Ctrl + C. OR Ctrl + Z.  
---------------------------------------------------------------------------
